# Student Productivity Hub

A modern full-stack MERN application designed to help students organize their study materials and track their progress effectively.

## Features

- **JWT-based Authentication**: Secure user signup and login.
- **Subject Management**: Users can add, view, and delete subjects with links to Google Drive folders.
- **Progress Tracking**: Log completed topics by date and chapter number.
- **Data Visualization**: Interactive charts (Bar, Line, Pie) to visualize progress, powered by Chart.js.
- **Modern UI**: A sleek, responsive interface built with React and Tailwind CSS, featuring glassmorphism and smooth animations.
- **RESTful API**: A robust backend built with Node.js, Express, and MongoDB.

## Tech Stack

- **Frontend**: React, Vite, Tailwind CSS, React Router, Axios, Chart.js
- **Backend**: Node.js, Express
- **Database**: MongoDB (with Mongoose)
- **Authentication**: JSON Web Tokens (JWT)

## Prerequisites

- [Node.js](https://nodejs.org/) (v18 or later recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)
- [MongoDB](https://www.mongodb.com/try/download/community) installed and running locally, or a MongoDB Atlas connection string.
